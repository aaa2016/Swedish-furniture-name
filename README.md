# Swedish Furniture Name Generator

## Purpose

This little tool takes a name provided by the user and: 
1. Reverses the letters.
2. The first instance in the name of 'A', 'O', or 'U' gets replaced with 'Ä', 'Ö', or 'Y', respectively.

For example - *Maria* becomes *Äiram*, *Simon* becomes *Nömis*

Note:
1. This does not work for palindromic names that don't contain any 'A', 'O', or 'U', eg. *Viv* or *Eve*.
2. 'QU', 'Q', and 'W' are replaced with 'KV', 'K', and 'V', respectively, due to being rare in Swedish words.
