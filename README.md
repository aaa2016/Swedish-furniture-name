# Swedish furniture name generator

## Purpose

This little tool takes a name provided by the user and: 
1. Reverses the letters
2. The first instance in the name of 'a', 'o', or 'u' get replaced with 'ä', 'ö', or 'y', respectively.

For example - *Maria* becomes *Äiram*, *Simon* becomes *Nömis*

Note: this does not work for palindromic names that don't contain any 'a', 'o', or 'u', eg. *Viv* or *Eve*.
